# Contributing guide

## Building

### Requirements

* gvp

### Setup environment

   ```sh
   mkdir -p GrimReaper/src/github.com/matee911 GrimReaper/bin GrimReaper/pkg
   cd GrimReaper/src/github.com/matee911
   git clone https://github.com/matee911/GrimReaper.git
   cd ../../..
   source gvp in
   ```

### Build

   ```sh
   go build
   ```
